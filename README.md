# mia-webhook-api
Node.js API Gateway - Messages webhook server

## Stack
- NodeJS
- Wit.ai
- FB Messenger

## Instruction
- Install packages

```
yarn
// or
npm install
```

- Build and Start Mia

```
yarn start
```

- If you wish to only build the chatbot itself. Use build command

```
yarn build
```

- Running Mia in dev mode

```
yarn dev
```
